# Cognitive Linguistics & Ethical Discourse Analysis
(comparison on discourse of Ashli Babbit and 

This project collects and analyzes official public statements referencing
Ashli Babbitt and Renee Good in order to examine how political authority,
agency, victimhood, and responsibility are linguistically constructed in
state and executive discourse.

The project is designed for academic use in ethics, political theory,
and cognitive linguistics.

## Research Questions
- How do official actors linguistically frame responsibility and agency?
- How does evaluative language differ across political administrations?
- What metaphors and narrative structures recur in official discourse?

## Methods
- Public-only data collection
- Source-limited scraping with respect for robots.txt
- Linguistic analysis using NLP techniques (spaCy, transformers)
- Reflexive documentation of limitations and bias

## Reproducibility
All scripts are version-controlled and documented.
Raw data is not redistributed where prohibited.
